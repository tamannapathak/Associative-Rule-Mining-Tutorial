# Associative-Rule-Mining-Tutorial

# How Interestingness Measures Shapes Association Rule Mining 

A tutorial on the practical importance of interestingness measure selection in association rule mining, developed as part of Msc Data Science at the University of Hertfordshire.


## Contents

tamanna_24175478.pdf
tamanna_24175478.ipynb

## What the Notebook Covers

1. Example of Associative Rule mining
2. Data loading and cleaning (UK transactions, top 100 items)
3. Basket matrix construction
4. Frequent itemset mining with FP Growth
5. Association rule generation
6. Computing five interestingness measures: lift, leverage, conviction, Kulczynski, cosine
7. Toy example demonstrating the null invariance problem
8. Empirical comparison: rank correlation heatmap, scatter plot 
9. Top rule profiles
10. When to use which measure
